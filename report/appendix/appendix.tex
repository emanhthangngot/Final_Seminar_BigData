\section{Phụ lục}

\subsection{Triển khai hệ thống benchmark}

Hệ thống được đóng gói bằng Docker Compose, cho phép khởi chạy toàn bộ bằng một lệnh duy nhất:

\begin{verbatim}
docker compose up -d
\end{verbatim}

Các dịch vụ bao gồm: Qdrant (cổng 6333), Weaviate (cổng 8080), Milvus Standalone (cổng 19530) cùng etcd và MinIO, Ollama (cổng 11434), FastAPI Backend (cổng 8000), và React Frontend (cổng 5173).

\subsection{Kiểm tra kết nối}

\begin{verbatim}
curl http://localhost:8000/api/v1/health
# {"status":"ok","databases":{"Qdrant":true,"Weaviate":true,"Milvus":true}}
\end{verbatim}

\subsection{Cấu hình Fairness Protocol}

Toàn bộ tham số benchmark được quản lý tập trung tại \texttt{src/config.py}:

\begin{verbatim}
HNSW_M = 16
HNSW_EF_CONSTRUCTION = 128
HNSW_EF_SEARCH = 64
DISTANCE_METRIC = "COSINE"
BENCH_CORPUS_SIZE = 10000
BENCH_NUM_QUERIES = 200
BENCH_SEED = 42
\end{verbatim}

Ba DB wrapper import các hằng số trên thay vì hardcode, đảm bảo tính công bằng.

\subsection{Mã nguồn và tài liệu}

Toàn bộ mã nguồn, slide thuyết trình, tài liệu benchmark và báo cáo được lưu trữ tại repository GitHub của nhóm.
